# TODO Tracker

> This file links to the live `TODO.md` at the repository root, where all in-flight work is tracked.

See [`../../TODO.md`](../../TODO.md) for the current Open / In Progress / Done lists.

## Quick reference

- **Open items** — work not yet started, assigned with owner and milestone target.
- **In Progress** — active tasks with a linked PR or branch.
- **Done** — completed and merged items, for audit trail.

The Director is the sole owner of `TODO.md` and updates it at every dispatch and completion event.
